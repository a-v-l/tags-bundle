## Requirements

Minimum requirements:

 - PHP 7.1
 - Contao 4.9
 - Haste 4.24

## Install using Composer

Execute the following command in your Contao 4.9 project folder:

    $ composer require codefog/tags-bundle

Then run the Contao install tool to update the database.
